# Instagram Reel + TikTok Script

**Format:** Screen recording with text overlays + voiceover or trending audio
**Length:** 15–30 seconds
**Goal:** Show the full user flow; make it feel satisfying and fast

---

## Recording Flow (what to capture)

Record these steps as one continuous screen recording, no cuts:

1. **Open the app** → dashboard appears, show monthly budget + remaining amount prominently
2. **Scroll** → show 2–3 categories with remaining budgets (e.g., Coffee: $12 left, Groceries: $47 left)
3. **Tap a category** → show transaction history + remaining budget for that category
4. **Tap "Add transaction"** → fill in amount and tap confirm
5. **Return to dashboard** → remaining budget visibly decreases (satisfying update)
6. **Zoom out on dashboard** → show the overall remaining budget bar

Total recording: aim for 20–25 seconds of raw footage, edit to 15–20s.

## Tips for Recording

- Use iOS/Android screen recording (no device frame needed for TikTok; optional for IG)
- Clean up test data before recording — use round numbers and realistic categories
- Record in good lighting if showing device, or use screen-only recording
- Do 3–4 takes, pick the smoothest one
