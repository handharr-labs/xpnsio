# Xpnsio Marketing Content

Promotion content across all platforms. All links should use UTM parameters.

## Current Version

**v0.3.0** — Latest release includes account deletion feature and centralized progress calculations.

## App URL

`https://xpnsio.vercel.app`

## UTM Parameters

| Platform | UTM string |
|----------|-----------|
| LinkedIn post | `?utm_source=linkedin&utm_medium=post` |
| LinkedIn article | `?utm_source=linkedin&utm_medium=article` |
| Twitter/X | `?utm_source=twitter&utm_medium=thread` |
| Threads | `?utm_source=threads&utm_medium=post` |
| Instagram carousel | `?utm_source=instagram&utm_medium=carousel` |
| Instagram reel | `?utm_source=instagram&utm_medium=reel` |
| TikTok | `?utm_source=tiktok&utm_medium=video` |
| Medium | `?utm_source=medium&utm_medium=article` |

## Key Messages (use on every platform)

- "Budget-first, not expense-first"
- "Know what you have LEFT — before you spend it"
- "Set budgets by category, by period (daily/weekly/monthly)"
- "Visual progress indicators: green (safe), yellow (warning), red (overrun)"
- "Your data, your control — full account deletion available"
- "Built by one person, currently free, no ads"

## Files

| File | Content |

| `long-form/linkedin-article.md` | Full LinkedIn article (~1500 words) |

| `long-form/medium-article-technical.md` | Medium article — technical deep-dive |

## 4-Week Posting Sequence

| Week | Platform | Content |
|------|----------|---------|
| 1 | — | Prepare screenshots, record demo, confirm app URL |
| 2 | LinkedIn feed | `short-form/linkedin-post.md` |
| 2 | Twitter/X or Threads | `short-form/twitter-thread.md` |
| 3 | Instagram | `visual/instagram-carousel.md` + optional Reel |
| 3 | TikTok | Repurpose Reel |
| 4 | Medium | `long-form/medium-article-product.md` |
| 4 | LinkedIn Articles | `long-form/linkedin-article.md` |
